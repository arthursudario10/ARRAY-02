# ARRAY-02
# Exercícios de Arrays em JavaScript

Repositório contendo exercícios desenvolvidos em JavaScript com foco em vetores (arrays), estruturas de repetição e lógica de programação.

## 📚 Atividades Desenvolvidas

### 1. Soma dos números pares
Programa que:
- lê 10 números inteiros;
- armazena em um vetor;
- calcula e exibe a soma apenas dos números pares.

---

### 2. Vetor com números aleatórios
Programa que:
- preenche automaticamente um vetor com 7 números aleatórios;
- exibe todos os valores gerados.

---

### 3. Contagem de ocorrências
Programa que:
- solicita 10 números ao usuário;
- armazena em um vetor;
- pede um número para busca;
- verifica quantas vezes ele aparece no vetor.

---

### 4. Remoção de valores duplicados
Programa que:
- lê 10 números inteiros;
- cria um novo vetor sem valores repetidos;
- exibe o vetor original e o vetor sem duplicatas.

---
